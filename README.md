## Estructura del Repositorio
- `/automatas/`: Contiene los archivos `.jff` de cada ejercicio, listos para ser importados y evaluados.
- `/imagenes/`: Recursos gráficos y diagramas utilizados para la compilación del documento.
- `*.tex`: Código fuente en LaTeX con el desarrollo matemático y razonamiento de cada máquina.
- `*.pdf`: Reporte final compilado, listo para su revisión directa.

## Ejecución y Pruebas
Los archivos `.jff` están diseñados para ejecutarse en **JFLAP**. Para verificar la integridad matemática de los AFND y AFD, abra el archivo correspondiente en la herramienta y utilice la función *Input > Multiple Run* para validar las cadenas de aceptación y rechazo documentadas en el PDF.
